Weight Loss Tracker

Welcome to theDrew's weight loss tracker. This page will track your weight from your starting point and graph your progress. All data is saved locally in your web browser, nothing is passed over the internet so be truthful when putting in your information. Though because all data is saved in the web browser, the information is not portable, you must use the same computer / browser to view your data all times or you will have to create a new user in each browser that you use.

This is just a beginning of a full application. there's still a good number of items that I need to work on before I can consider myself done, like:

  • The 'Edit Users' option up there
  • The 'Help' Option up there
  • I ask you your "Goal Weight" but then never do anything with it.
  • I did media queries for different screen sizes, but it would be really nice if the chart changed sizes when screen did. [I'm using JQPlot for the charting and this might be a limitation of the tool I'm using, But I'll do some more research.]
  • Over all the look is nice, but not the 'great' look and feel that I'm going for.
  • I was originally planning on including more data such as BMI and Body Fat that you could optionally track, but it was left out of the first version.
  • I need to work on a better UI for the "Weigh In" entry. It just looks kinda sad down there.
If there is anything else that you think I missed or that which could be better, please drop me a note and let me know. Thanks.
